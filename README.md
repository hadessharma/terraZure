# TerraZure

Manage your Azure cloud infrastructure and deployments with ease.

## Features

- **Subscription Management**: Track and organize your Azure subscriptions.
- **Resource Overview**: Visualize and monitor your cloud resources.
- **Simple Dashboard**: A clean interface to manage your infrastructure.

## Getting Started

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/hadessharma/terraZure.git
    cd terraZure
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Set up environment variables:**
    
    Create a `.env` file in the root directory and add your configuration (database connection, authentication keys, etc.).

4.  **Run the application:**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
